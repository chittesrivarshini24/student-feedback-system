# SMART ONLINE SURVEY SYSTEM

## Project Overview

SMART ONLINE SURVEY SYSTEM is a web-based feedback collection and analytics platform developed using HTML, CSS, JavaScript, Firebase Authentication, and Firebase Firestore.

The system allows users to register, log in, submit surveys, and view feedback analytics through charts and statistics.

## Features

### User Authentication
- User Registration
- User Login
- Firebase Authentication
- Logout Functionality

### Survey Types
- Student Feedback Survey
- Hospital Feedback Survey
- Employee Feedback Survey
- Office Feedback Survey

### Analytics Dashboard
- Total Responses Count
- Survey Statistics
- Average Satisfaction Score
- Bar Chart Visualization
- Pie Chart Visualization

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Firebase Authentication
- Firebase Firestore
- Chart.js

## Working

1. User registers an account.
2. User logs into the system.
3. User selects a survey type.
4. User submits feedback.
5. Feedback is stored in Firebase Firestore.
6. Analytics dashboard displays survey statistics.
